# Helm Charts for Chatwoot

---
This repository contains helm charts for [Chatwoot](https://chatwoot.com).

## Installation

```bash
helm repo add maxcrm https://ittron.github.io/maxcrm-charts
helm install maxcrm chatwoot/chatwoot
```
