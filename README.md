# Helm Charts for Chatwoot

---
This repository contains helm charts for [Chatwoot](https://chatwoot.com).

## Installation

```bash
helm repo add chatwoot https://chatwoot.github.io/charts
helm install chatwoot chatwoot/chatwoot
```
